# E-Commerce Store

Modern e-commerce web application built with React, TypeScript, Redux Toolkit and Material UI.

## Features

- Product catalog
- Product details page
- Search products
- Category filtering
- Favorites system
- Shopping cart
- Quantity management
- LocalStorage persistence
- Dark / Light theme
- Responsive design
- Mobile drawer navigation
- Skeleton loaders
- 404 page

## Technologies

- React
- TypeScript
- Vite
- Redux Toolkit
- React Router
- Material UI (MUI)
- Fake Store API

## Installation

```bash
git clone <repository-url>
cd ecommerce-store
npm install
npm run dev
```

## Build

```bash
npm run build
```
